# 🚀 DevHub – Collaborative Developer Platform

DevHub is a full-stack SaaS platform where teams collaborate using a real-time Kanban board.

---

## ✨ Features

- Authentication with NextAuth
- Project & Team Management
- Kanban Task System
- Comments & Discussions
- Search Projects
- Role-Based Access

---

## 🏗 Tech Stack

**Frontend**
- Next.js (App Router)
- Tailwind CSS

**Backend**
- Server Actions
- PostgreSQL (Neon)
- Prisma ORM
- Raw SQL queries

---

## 📂 Project Structure

- src/ → Application code
- docs/ → Technical documentation
- prisma/ → Database schema

---
